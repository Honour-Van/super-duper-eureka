# Rules for Naming Files

File "Movement_vi_k_ap.csv" means that this .csv file is the "k"th data collected with actions of "Movement" by the collector "vi"

Example:
* "SitStand_v1_3_ap.csv" means that 
  
  * action: sit and stand
  * collector: v1
  * version: 3
